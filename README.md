# My PhotographySite 

Created by Vite3 Vue3 tailwindcss 🌍

## Getting started

### 1.Clone this project
```
$ git clone https://github.com/Ekarmore/Photography-Site.git
```
### 2.Enter the project directory and install dependencies
```
$ npm install
```
### 3.Develop the project
```
# npm
$ npm run dev
# yarn
$ yarn dev
```
### 4.Build
```
# npm 
$ npm run build
# yarn
$ yarn build
```
